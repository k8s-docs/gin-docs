## 什么是 Gin?

Gin 是用 Go 写的一个 HTTP Web 框架 (Golang).
它具有一个有更好的性能 Martini-like API -- 高达 40 倍的速度.
如果您需要砸性能, 让自己的一些 Gin.

## 如何使用 Gin?

我们提供了一些 API 使用[实例](https://github.com/gin-gonic/examples)并列出了一些大众所熟知[Gin 的用户](./users).
